Returns module specified in app.json for a slot

Searches the app.json for the module meant to fill the slot {slot} of module {parent_module}.  Creates and returns this module, or null if the slot was not filled.

Parameters

parent_moduleModule for which to create submodule
slotSlot for which to create module
extra_propsdictionary of construct properties for the submodule

reference_slot,

Passes an existing module instance through the callback

Passes an existing module instance through the callback if the module instance is available, if is not available, just keeps the callback so it can be called later when the module is created.

Parameters

modulemodule asking for another module instance
reference_slotreference slot which contains the id of module instance
callbackfunction to be called whenever the instance is available
