EknWebview

WebKit WebView subclass which provides utility functions for loading and handling content from the knowledge engine and a clean interface for injecting custom JS and CSS from gresource files.

Calling <inject_js_from_resource()> or <inject_css_from_resource()> will ensure that the current page (if any) as well as all subsequently loaded pages in this webview will have the given JS or CSS injected only when the HTML document has finished loading

Parent class
