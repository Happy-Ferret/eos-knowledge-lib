function get_data_dir(app_id)

Searches for the EKN manifest for a knowledge engine domain

Parameters

app_idknowledge app ID, such as “com.endlessm.health-es”.

This function searches through the system data directories for an EKN data directory for the given domain.

Returns

a Gio.File pointing to an app content directory, or null.
