Properties

child-tags

A list of tags that articles in this set are tagged with

child-tags refers to the articles that are contained within a set; those articles are all articles whose ContentObjectModel.tags properties contain one of the tags in the set’s child-tags property.

Note that ContentObjectModel.tags on a SetObjectModel refers to the tags with which a set itself has been tagged.
