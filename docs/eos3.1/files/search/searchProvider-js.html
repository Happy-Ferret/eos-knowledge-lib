SearchProvider

A search provider for a single knowledge app, to be used through dbus by the shell’s global search.  Requires the app id of the knowledge app it should run searches for.

This search provider will activate the actual knowledge app over dbus with a result or search to display.  As such, it can be used from in or outside of the actual application process.
