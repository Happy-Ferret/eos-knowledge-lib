Properties

model

Record backing this card

Every card is backed by a record in the database.  A card’s record is represented by a <ContentObjectModel> or one of its subclasses.

Type

<ContentObjectModel>

Flags

Construct only

highlight-string

A substring within a card’s title or synopsis to get highlighted

Sometimes we want to highlight a particular substring with the text of a card.  This property specifies which substring to highlight.

text-halign

Horizontal alignment of text when card is in not in horizontal mode.

Default value

**Gtk.Align.CENTER**

sequence

A Sequence value for the card.  Previous or next.

background-size

How to scale the background thumbnail image.

This property determines how we scale the thumbnail image.  It uses a similar vocabulary as web css’ background-size property: ‘cover’ and ‘center’.  ‘cover’ means we will reposition and rescale the image to cover the entire thumbnail space.  ‘center’ means we will maintain the image’s original dimensions and center it in the thumbnail space.

create_context_widget_from_model

create_context_widget_from_model: function ()

Sets the text on a context label.  The context is the list of tags associated with this article.  Tags are also incidentally the same as set titles.

According to designs (and based on what fits on screen), we only want to show a maximum of two set titles in the context label.  Moreover, we want to exclude the ‘Ekn’-prefixed tags which won’t mean anything to the user.
