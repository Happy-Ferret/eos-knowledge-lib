SpaceContainer

Container that shows only as many widgets as it has space for

Usually a container will try to display all the widgets that have been added to it.  You can add as many widgets as you want to SpaceContainer, but it will only show as many as it has space for.  If you resize your window so that there is space for more widgets, they will appear.
