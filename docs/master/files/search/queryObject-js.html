QueryObjectType

INCREMENTALQueries will match partially typed words.  For example a query for ‘dragonba’ will match the Dragonball article.
DELIMITEDQueries will be assumed to be entire words.

QueryObjectMatch

TITLE_ONLYOnly article titles will match against the query string.
TITLE_SYNOPSISArticle titles and synopsis will match the query string.

QueryObjectSort

RELEVANCEUse xapian relevance ranking to sort articles.  Exact title matches will be weighted most heavily.
SEQUENCE_NUMBERUses the article page rank to sort results.

QueryObjectOrder

ASCENDINGReturn articles in ascending order.
DESCENDINGReturn articles in descending order.

QueryObject

The QueryObject class allows you to describe a query to a knowledge app database for articles.  Use this with Engine.get_object_by_id to retrieve article from the database.

This class has no functionality, but is just a bag of properties to tweak the type of query being made.  QueryObjects are immutable after creation, which allows them to be used safely in a HistoryItem.  All properties must be passed in on construction.

See new_from_object for a convenience constructor to create a new object with a few tweaked values.

Properties

app-id

The app ID of the database to query.  If not set, the Engine will fill this in with the Engine.default-app-id property.

query

The actual query string that was entered by the user with all terms that should be searched for.

stopword-free-query

A corrected version of the query property with stopword words removed.

type

The type of query to preform, see QueryObjectType.

Defaults to QueryObjectType.INCREMENTAL.

match

What to match against in the source documents, see QueryObjectMatch.

Defaults to QueryObjectMatch.TITLE_ONLY.

limit

The maximum number of results to return.

Defaults to 10.

offset

The number of results to skip, can be use with limit to paginate results.

Defaults to 0.

sort

What to sort results by, see QueryObjectSort.

Defaults to QueryObjectSort.RELEVANCE.

order

Order of results, see QueryObjectOrder.

Defaults to QueryObjectOrder.ASCENDING.

tags_match_all

A list of tags to restrict the search to.

tags_match_any

A list of tags to restrict the search to.

ids

A list of specific ekn ids to limit the search to.  Can be used with an empty query to retrieve the given set of ids.

excluded_ids

A list of specific ekn ids to exclude the search from.

A list of specific ekn ids to exclude the search from.
