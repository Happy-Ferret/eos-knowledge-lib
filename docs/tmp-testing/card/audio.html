Card.Audio

Card.Audio

This widget displays the title if enabled via properties, the synopsis if included in the object model, and the player for reproducing the audio.

Style classes: CardAudio - on the widget itself CardAudio__title - on the title label CardAudio__synopsis - on the synopsis label CardAudio__player - on the audio player
